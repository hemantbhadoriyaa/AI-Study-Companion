# Hemantooo AI Study Companion

Hemantooo is an AI-powered desktop study assistant that uses a webcam to track focus, detect distraction, and improve productivity using real-time computer vision and the Pomodoro technique.

It also generates a detailed PDF report of each study session.

---

## Features

- Real-time face and eye tracking using OpenCV and MediaPipe  
- Focus vs distraction detection  
- Drowsiness detection system  
- Pomodoro mode (focus and break cycles)  
- Voice alerts for attention control  
- Live productivity tracking  
- Automatic PDF study report generation  
- Session-based performance analysis  

---

## Tech Stack

- Python 3  
- OpenCV  
- MediaPipe  
- Tkinter (GUI)  
- NumPy  
- Pillow  
- ReportLab
